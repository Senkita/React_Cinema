# React_Cinema

## Introduction

这里就写了个播放器，别的啥也没有。

服务器上挂了 Aria2 用于下载网盘视频，下载好的视频想直接在线播放，所以就摸出了这个项目。

## Usage

```bash
cd React_Cinema
pnpm i

pnpm dev # `pnpm build` for production
```
